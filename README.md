# dotfiles

personal setups and other things

# Installation

0. Prereq
- Homebrew
- Homebrew brew 
- Stow (required for symlinking dotfiles)


1. Clone the repo
```
https://github.com/hbish/dotfiles.git ~/.dotfiles
```

2. Make zsh as default shell
```
chsh -s /usr/local/bin/zsh
```

3. Install prezto

4. Symlink
```
cd ~/.dotfiles
stow <folder_name>
```
